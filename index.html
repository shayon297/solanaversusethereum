Real Economic Value (REV)

Solana has consistently generated more REV than Ethereum since November 2024

90-Day Rolling Average

Definition: REV refers to Real Economic Value, or the total amount users are willing to pay to transact on a blockchain during a given period.

Methodology: Solana REV: Base Fees + Priority Fees + Vote Fees + Jito Tips; Ethereum REV: Base Fees + Priority Fees + MEV-Boost Tips

Source: Blockworks

Average Transaction Cost

Methodology: Daily transaction fees divided by daily transaction count for each blockchain

Source: TokenTerminal
